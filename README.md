## 🛠 技术架构

### 🧠 AI核心引擎
```python
# 核心检测流程
class AllergenDetector:
    def scan_ingredients(self, image):
        # 1. OCR文字提取
        ingredients = self.ocr_engine.extract_text(image)
        
        # 2. 成分标准化
        normalized = self.normalizer.process(ingredients)
        
        # 3. 过敏原匹配
        allergens = self.matcher.find_allergens(normalized)
        
        # 4. 风险评估
        risk_level = self.risk_assessor.evaluate(allergens)
        
        return {
            "allergens": allergens,
            "risk_level": risk_level,
            "safe": risk_level == "low"
        }
```
